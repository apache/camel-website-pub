Apache camel-k 2.1.0 Release

New and Noteworthy

Supported Java version

This version supports Java 17.

Apache Camel-K

Download Signature and checksum
camel-k-dist-2.1.0 (CLIs Distribution) PGP Signature, SHA512 Checksum
camel-k-sources-2.1.0.tar.gz (Sources) PGP Signature, SHA512 Checksum

Git tag checkout

Release is tagged with v2.1.0 in the Git, to fetch it use: 

git clone https://git-wip-us.apache.org/repos/asf/camel-k.git
cd camel-k
git checkout v2.1.0
