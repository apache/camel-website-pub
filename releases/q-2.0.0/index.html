New and Noteworthy

Supported Java version

This version supports Java 11.

Apache Camel Quarkus

Download Signature and checksum
apache-camel-quarkus-2.0.0-src.zip (Sources) PGP Signature, SHA512 Checksum

Git tag checkout

Release is tagged with 2.0.0 in the Git, to fetch it use: 

git clone https://git-wip-us.apache.org/repos/asf/camel-quarkus.git
cd camel-quarkus
git checkout 2.0.0
