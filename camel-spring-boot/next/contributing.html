Contributing

Building the code

You need Apache Maven version 3.9.0 or greater to build the code. You can either use your own Maven package or build using the Maven wrapper (mvnw) provided with the project.

mvn -Pfastinstall clean install

Reporting a bug or problem

The Camel Spring Boot project uses the Apache Foundation Jira instance for tracking issues, tasks, and feature requests.
